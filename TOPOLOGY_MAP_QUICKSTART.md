## 5-Minute Quick Start

### **Step 1: Launch the App**
1. Run your ClusterView application
2. Navigate to the **Topology Map** view
3. You'll see clusters displayed in a circular layout

### **Step 2: Explore with Hover**
1. **Move your mouse** over any cluster node
2. Watch the edges (lines) between clusters **brighten**
3. The node's border becomes **more prominent**
4. This is a **preview** - no commitment needed

### **Step 3: Select a Cluster**
1. **Click on any cluster** you're interested in
2. The node turns **bright coral color**
3. A **detail panel** slides in from the right side
4. Now you can see detailed information about this cluster

### **Step 4: Read the Details**
In the detail panel, you'll see:
- Cluster name (uppercase)
- Number of files
- Directory path
- Last modification date
- Whether duplicates exist
- List of all related clusters

### **Step 5: Navigate Between Clusters**
1. In the detail panel, scroll down to see **"Related Clusters"**
2. **Click on any cluster name** to switch to it
3. The panel updates immediately
4. Click the **X button** to close the panel

### **Step 6: Control the View**
- **Drag canvas** = Pan around
- **Scroll mouse wheel** = Zoom in/out
- **Click Reset button** = Return to default view
- **Drag cluster node** = Move it to a new position

### **Step 7: Hide Clusters**
1. In the detail panel, find the **"Related Clusters"** section
2. Look for the **eye icon** (üëÅÔ∏è) next to each cluster
3. **Click the eye** to hide/show clusters
4. This reduces visual clutter

### **Step 8: Get Help**
- Look at the **bottom-left corner** for the **CONTROLS legend**
- It shows all available actions at a glance:
  - Drag ‚Üí Pan view
  - Scroll ‚Üí Zoom
  - Click Node ‚Üí Select
  - Drag Node ‚Üí Move

```

```

### **"How do I go back to the main dashboard?"**
```
1. Click [‚Üê BACK] button in top-left
```

## Visual Indicators Explained

### **Cluster Node Colors**
- üîµ **Blue** = Default, not selected
- üî∂ **Coral** = Currently selected
- Gray border = Not being interacted with
- Bright border = Being hovered or selected

### **Connecting Lines**
- ‚ûñ **Faint line** = Default connection (subtle)
- ‚ûñ **Medium line** = Node being hovered (brighter)
- ‚ûñ **Bright line** = Node is selected (brightest)

### **Detail Panel**
- Appears on **right side** when you click a cluster
- Slides in smoothly
- Shows all information about selected cluster
- Click X to close it

### **Toolbar**
- At the **top** of the map
- Shows current **Zoom level** (e.g., "Zoom: 1.5√ó")
- Has **Reset** and **Back** buttons

## Tips & Tricks

### **Pro Tip #1: Explore Visually**
Hover before clicking! This lets you see all connections without committing to selection.

### **Pro Tip #2: Focus Your View**
Use visibility toggles to hide irrelevant clusters and focus on what matters.

### **Pro Tip #3: Find Positions**
Arrange clusters manually by dragging them, then use Reset if you want to go back.

### **Pro Tip #4: Quick Navigation**
The Related Clusters list is the fastest way to jump between clusters. No clicking back to dashboard needed!

### **Pro Tip #5: Use the Legend**
Don't remember the controls? They're always visible in the bottom-left corner!

## Troubleshooting

### **"A cluster disappeared!"**
- Check if you accidentally hid it
- Go to detail panel and toggle eye icons
- Click Reset to restore default view

### **"I can't see the detail panel"**
- Make sure you clicked on a cluster node
- Try clicking a different cluster
- Make sure nothing is covering the right side

### **"The zoom is stuck"**
- Click the Reset button
- Or try scrolling in the opposite direction

### **"Clusters are scattered all over"**
- You probably dragged them!
- Click Reset to restore the circle layout

### **"Controls legend is in the way"**
- It's semi-transparent, so you can still see the map behind it
- You can move your view by panning or zooming
